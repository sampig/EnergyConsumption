Energy Consumption

Data Format

Information contained in QR Code. There are 3 types:

  1. Plain Text:
    1. only Device ID
    2. only URL (start with http://)
  2. JSON Text: {'deviceID':'device_id','webserver':'url','direct':boolean}

Response Data: ecws/deviceID/{deviceID}[/quantity]

{
    'DeviceID':'device_id',
    'Data':{
        '2016-06-29 12:36:25':33.3137,
        'time':value
    }
}
